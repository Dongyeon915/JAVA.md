# if 문
````
조건식에는 true 또는 false 값을 산출할 수 있는 연산식이나, boolean변수가 올수 있다.
조건식이 true이면 블록을 실행하고 false이면 블록을 실행하지 않는다
````

# if-else 문
````
if문은 else 블록과 함께 사용되어 조건식의 결과에 따라 실행블록을 선택 할수있다.
조건식이 true일경우 if문의 블록이 실행되고, 조건식이 false이면 else 블록이 실행된다
조건식의따라 하나의 블록만을 실행하며 if문을 벗어나게된다
````

# if-else if-else 문
````
처음 if문의 조건식이 false일 경우 if 블록의 끝에 else if문을 붙이게되면 여러개의 조건중 true가 되는 블록만 실행하고 if문을 벗어나게된다
else if문의 수는 제한이 없다
````
# 중첩 if 문
````
if문 블록 내부에는 또 다른 if문 사용이 가능하다
중첩 단계는 제한이 없고 실행 흐름에 맞게 작성하면된다.
중첩 if문은 조건식이 true라면 좀더 세부적인 기준을 작성할수있다
````

# Switch 문
````
변수가 어떤 값을 갖느냐에 딸라 실행문이 선택된다.
case끝에 break가 붙은 이유는 다음 case를 실행하지 말고 switch문을 빠져나가기 위함이다.

자바 6까지는 byte,char,short,int,long 변수나 정산값을 산출하는 연산식만 올수 있었지만 자바 7부터는 String 타입의 변수도 올 수 있다.
````

# for 문 (반복문)
````
주어진 횟수만큼 실행문을 반복 실행할 때 적합한 반복 제어문이다.
1. 초기화식이 제일 먼저 실행된다.
2. 조건식을 평가해서 true이면 (3)실행문을 실행한다
3. false 일경우 for문 블록을 실행하지 않고 끝나게 된다
4. 증감식을 실행시키고 다시 (2) 조건식을 평가하게 된다.
5. 평가 결과가 true라면 실행->증감->조건 식으로 다시 진행되게 된다

초기화식과 증감식은 둘이상 올수있다
for(int i=0, j=0; i<=50 && j>=50; i++, j--)

부동소수점 타입은 사용하지 말아야한다
타입을 정확하게 표현할 수 없기 때문에 루프 회수보다 적게 실행된다.
````

# 중첩 for 문
````
바깥쪽 for문이 한 번 실행할 때마다 중첩된 for문은 지정된 횟수만큼 반복해서 돌다가 다시 바깥쪽 for문으로 돌아간다.
````

# while 문
````
조건식이 true일 경우에 계속해서 반복한다
조건식으로는 비교 또는 논리 연산식이 주로 오며 false가 되면 반복 행위를 멈추고 종료한다
````

# do-while 문
````
블록 내부의 실행문을 우선 실행시키고 실행 결과에 따라서 반복 실행을 계속할지 결정이 가능하다
1. 실행문 실행
2. 조건식 평가
3. true일 경우 1 -> 2 반복실행 
4. false라면 종료
````

# break문
````
반복문 종료시 이름:for(~~) 이름부분을 
break 이름; 으로 반복문 마지막에 작성한다면 해당 반복문을 종료할수있다.
````

# continue 문
````
반복문 for,while,do-while에서만 사용되며 continue문 실행시 for문은 증감식,while문은 조건식으로 이동한다

반복문을 종료하지 않고 계속 반복을 수행한다
if문에 같이 사용되며 특정 조건을 만족하는 경우 그이후의 문장을 실행하지 않고 다음 반복으로 넘어간다.
````
